오늘 목표 100% 달성하기 - To Do List

사용된 기술들

리액트와 함께하는 첫 프로젝트로 useState를 이용하여 초기 데이터의 CRUD를 다루어 보았습니다. 배열 상태의 데이터를 리액트 불변성을 유지하면서 id 값을 확인하여 필요한 데이터만을 관리하는 방법에 대하여 연습할 수 있었습니다. 또한, Filter 객체를 만들어 사용자가 선택한 Filter의 값에 맞는 데이터만을 화면에 구성하고, 데이터의 수정 모드를 확인하여 모드에 따라 다른 컴포넌트를 마운트 할 수 있도록 구성해 보았습니다.

하나의 js 파일에서 컴포넌트와 스타일을 모두 다룰 수 있는 tailwind css를 리액트에 적용시켜 사용하였습니다. 스타일을 적용/수정할 때 파일을 왔다 갔다 반복할 필요가 없어 간편했습니다. [] 안에 임의의 값을 넣어 tailwind가 제공하지 않는 속성을 생성하여 적용해 보았습니다.

적용된 기능들

  • 화면 상단에서 윈도우 설정값에 따른 날짜 확인이 가능합니다.

  • 목표 달성 여부를 토글 하면 달성률을 계산하여 애니메이션과 함께 값이 변경됩니다. 목표 label이나 checkbox를 클릭하면 토글 됩니다.

  • 입력창에 데이터를 입력하면 새로운 목표가 목록 가장 하단에 나타납니다.

  • 전체 & 진행 중 & 완료 버튼을 클릭하면 현재 목록의 목표 중 버튼의 값에 해당되는 데이터만 필터링하여 보여줍니다. 기존의 목표가 수정 중일 때는 동작하지 않습니다.

  • 목표의 화살표 버튼을 클릭하면 내용 수정이 가능하고 이전 목표를 placeholder에 띄워줍니다. 수정 모드에서는 이전으로 돌아가기 버튼과 확인 버튼이 있고, 값을 수정하지 않으면 확인 버튼은 비활성화됩니다.

  • 휴지통 버튼을 클릭하면 오른쪽에서 왼쪽으로 element가 사라지며 데이터를 삭제합니다. 완료된 목표를 삭제하는 경우 달성률도 변경됩니다.
