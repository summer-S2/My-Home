ABOUT ME

안녕하세요! 프론트엔드 개발자를 꿈꾸는 정윤지입니다.


다양한 경험 그리고 새로운 시도를 좋아합니다.

평소에 웹사이트를 이용하면서 '이런 기능이 있으면 좋을텐데..'하고 생각만 했던 기술을 스스로 고민하고 구현해보면서 큰 성취감을 느꼈습니다.

웹개발의 가장 큰 매력은 무궁무진한 가능성이라고 생각합니다.


다양한 사람들이 만족할 수 있는 웹페이지를 개발하고 꾸준히 소통하는 개발자가 되고싶습니다.

MY SKILLS

HTML

웹표준과 웹접근성에 대해 이해하였고, 누구나 쉽게 접근하고 상호작용할 수 있는 마크업 언어를 학습하였습니다.

CSS

사용자 친화적인 UI를 제공하기 위해 요소의 속성과 속성값, 작동원리를 이해하였고, 실제로 구현하도록 노력하였습니다.

JAVASCRIPT

REACT

리액트 불변성과 컴포넌트의 재사용성을 이해하고, 렌더링이 최적화된 웹앱을 구현하도록 노력하였습니다.

TAILWIND

직관적이고 자유도가 높으며, 빠른 스타일링이 가능한 CSS 프레임워크를 학습하였습니다.
